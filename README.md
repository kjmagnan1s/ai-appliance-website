# AIppliance Manager Web

Marketing site for the AIppliance Manager mobile app.

## Development

```sh
npm install
npm run dev
```

The site is built with Vite, React, TypeScript, Tailwind CSS, and shadcn/ui.

## Production Build

```sh
npm run build
npm run preview
```

Deploy the contents of the `dist` folder to your static hosting provider of choice.
